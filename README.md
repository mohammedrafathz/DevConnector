# DevConnector
A social networking website for developers.

clone the repo & install the node_modules for server and client
# Clone
```
> git clone https://github.com/risky-boy/DevConnector.git
```
# Server
```
> cd DevConnector
> npm install
> npm start server
```

# Client
```
> cd DevConnector/client
> npm install
> npm run start
```
